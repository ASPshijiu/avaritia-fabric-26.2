# 更新日志

## 1.0.0-rc.1 - 2026-07-21

首个功能完整的 Minecraft Java 26.2 Fabric 发布候选版。

### 新增

- 完整材料成长链、14 个默认奇点与 13 个条件式科技奇点。
- 压缩工作台、二重压缩工作台、9×9 终极工作台及其有序/无序配方。
- 四级中子收集器、四级中子压缩机、压缩箱子与无尽箱子。
- 经典终局武器、工具、护甲、食物，以及功能矩阵中列出的 Re:Avaritia 扩展内容。
- 数据驱动奇点、压缩配方、双语资源与配方进度解锁链。
- OpenGL 与 Minecraft 26.2 实验性 Vulkan 后端兼容。

### 工程质量

- 88 个 GameTest 覆盖合成、使用、能力边界、保存、同步和机器性能。
- 构建期资源一致性门禁覆盖 568 个 JSON、96 个配方、27 个可掉落方块和 208 个双语键。
- 完成真实发行 JAR 的客户端、单人世界、专用服务器首次启动/重启及单客户端多人重连验证。

### 兼容性

- 仅支持 Minecraft Java 26.2、Java 25、Fabric Loader 0.19.3 与 Fabric API 0.155.2+26.2 基线。
- 使用新的 `avaritia26` 命名空间，不提供旧存档迁移。

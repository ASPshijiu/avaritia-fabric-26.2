# Avaritia 26 (Unofficial)

面向 Minecraft Java 26.2 的非官方 Fabric 无尽贪婪适配。

> 当前状态：开发中，尚未发布可玩版本。

## 开发基线

- Minecraft 26.2
- Java 25
- Fabric Loader 0.19.3
- Fabric API 0.155.2+26.2
- Fabric Loom 1.17.16
- Gradle 9.5.1

## 构建

```bash
./gradlew build
```

完整范围、阶段和验收门禁见 [DEVELOPMENT_PLAN.md](DEVELOPMENT_PLAN.md)。

## 项目身份

本项目是社区维护的非官方适配，与 Avaritia 原作者及其他移植项目没有官方关系。

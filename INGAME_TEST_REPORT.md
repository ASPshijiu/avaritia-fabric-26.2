# 真实游戏内逐项验收记录

本报告只记录真实 Minecraft 26.2 客户端中的键鼠操作、画面观察和对应运行日志。GameTest 结果不能代替这里的人工操作结果。

## 验收环境

- Minecraft Java 26.2
- Java 25.0.3
- Fabric Loader 0.19.3
- Fabric API 0.155.2+26.2
- 模组：`1.0.0-rc.1` 发行 JAR
- 客户端：macOS 26.5.2、Apple M5、OpenGL 4.1 Metal
- 世界：创造模式单人验收世界；需要生存语义时显式切换游戏模式

## 状态定义

- `通过`：已经在真实客户端中覆盖该物品当前全部适用行为并观察结果。
- `进行中`：已经完成部分真实客户端操作，但仍有适用行为尚未实测。
- `失败`：已经复现不符合预期的行为，必须先修复。
- `待复测`：问题已修复，但尚未在真实客户端重新操作。
- `待实测`：只有自动化或源码证据，尚未进行人工操作。
- `不适用`：该物品没有对应行为，并在备注中说明原因。
